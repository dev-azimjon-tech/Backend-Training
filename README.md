# Backend-Training

A repository for honing backend development skills using Python. This project encompasses a variety of console-based applications and web services built with the Flask framework.

## Overview

This project serves as a practical training ground for backend engineering concepts. It is designed to build proficiency in Python, API development, and the fundamentals of web frameworks.

## Features

*   **Console Applications:** A collection of command-line tools demonstrating core Python principles.
*   **Flask Web Services:** RESTful APIs and web applications built with Flask.

## Technologies

*   **Language:** Python 3
*   **Framework:** Flask

## Getting Started

### Prerequisites

*   Python 3.x
*   pip (Python package installer)

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/Backend-Training.git
    cd Backend-Training
    ```

2.  **Create and activate a virtual environment:**
    ```bash
    # For Windows
    python -m venv venv
    .\venv\Scripts\activate

    # For macOS/Linux
    python3 -m venv venv
    source venv/bin/activate
    ```

3.  **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

## Usage

Each application or service is contained within its own directory. Please refer to the `README.md` within each respective directory for specific instructions on how to run it.
